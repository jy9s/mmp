# Marketing Attribution Dashboard

## Project Motivation

広告データを分析する際に、複数のツールを使う必要があり不便でした。

そこで、キャンペーンのパフォーマンスを一つの画面で確認できる
ダッシュボードを開発しました。

---

## Tech Stack

Java  
Spring Boot  
JPA  
MySQL  
Thymeleaf  

---

## Main Functions

### Campaign Management

- Create campaign
- Update campaign
- Delete campaign

### Performance Tracking

- impressions
- clicks
- conversions
- cost

### KPI Calculation

- CTR
- CVR
- CPC
- CPA

---

## Lessons Learned

Spring Bootを使ったWebアプリケーションの設計を学びました。

また、JPAを使用する際のパフォーマンス問題について理解を深めました。
